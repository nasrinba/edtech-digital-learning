The difference between CSS Grid Layout and CSS Flexbox Layout is that flexbox was designed for layout in one dimention - either a row or a column.
Grid was designed for two-dimentional layout - rows, and columns at the same time.

Bootstrap is the most popular HTML, CSS and Javascript framework for building responsive, mobile-first projects on the web. Tailwind CSS, on the other hand , is the most popular utility-first CSS framework for fast UI development.
Bootstrap is a component-based framework, which means it comes with prebuilt components and includes other utilities for layering displays, spacing, etc.
Tailwind, on the other hand, CSS is a utility-first framework. Using Tailwind CSS is a kin writing regular CSS. Unlike Bootstrap, it has no prebuilt compunents.

A CSS box model is a compartment that includes numerous assets, such as edge, border, padding and material. It is used to develop the design and structure of a web page.
It can be used as a ser of tools to personalize the layout of different components. According to the CSS box model, the web browser supplies each element as a square prism.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.
Some Semantic tags, such as- header, nav, section, article, aside, footer etc.
